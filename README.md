## Project Overview

TrackBP is an activities tracking application being developed as a bachelor's thesis project.

## Repository Structure

```
/server          - Backend application
  /src           - Source code
  /tests         - Test files
  /docs          - Documentation
  /bruno         - Bruno API collection for testing endpoints
```

## Development Commands

Commands to be added once package.json is configured. Expected patterns:

- `npm install` - Install dependencies
- `npm run dev` - Start development server
- `npm test` - Run tests
- `npm run test:watch` - Run tests in watch mode

## Environment

The server uses environment variables stored in `server/.env`.
